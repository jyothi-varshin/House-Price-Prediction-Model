# House Price Prediction

## Overview
This project is a **regression-based machine learning model** designed to predict house prices using features like **location, area, number of rooms**, and more.  
It demonstrates **data preprocessing, visualization, and model training** using Python libraries.  

The model helps in understanding how different features influence house prices and can be extended for real-world applications.  

---

## Features
- Data preprocessing:
  - Handling missing values
  - Encoding categorical variables with One-Hot Encoding
- Exploratory Data Analysis (EDA) with visualizations using **Matplotlib** and **Seaborn**
- Regression model training using **Linear Regression**
- Model evaluation with metrics such as **Mean Absolute Percentage Error (MAPE)**
